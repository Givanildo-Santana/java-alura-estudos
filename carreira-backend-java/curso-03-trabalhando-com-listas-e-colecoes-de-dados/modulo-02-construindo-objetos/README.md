# Módulo 02 — Construindo objetos

Este módulo apresenta o uso de construtores em Java para inicializar objetos com parâmetros.

---

## O que foi aprendido

- Passagem de parâmetros ao instanciar objetos com construtores;
- Como funciona o construtor padrão (default) no Java;
- Como declarar construtores em uma classe;
- Chamada do construtor da classe mãe com `super`.

---

## Estrutura do módulo

- `projeto-screenmatch/` — construtores adicionados em `Titulo`, `Filme` e `Serie`; `Principal` atualizado para instanciar com `new Filme("nome", ano)`
- `desafios/` — exercício prático com `Produto` e `ProdutoPerecivel`

---

## Desafio

| Pacote | Classes | Descrição |
|---|---|---|
| `modulo02.desafios` | `Produto`, `ProdutoPerecivel`, `Principal` | Lista de compras com construtor, herança, validação nos setters e polimorfismo na ArrayList |

---

## Como executar

1. Abra o projeto na sua IDE (IntelliJ IDEA, por exemplo).
2. Navegue até:

   `curso-03-trabalhando-com-listas-e-colecoes-de-dados/src/`

3. Localize a classe `Principal` do pacote desejado.
4. Execute a aplicação (botão de **Run**).
