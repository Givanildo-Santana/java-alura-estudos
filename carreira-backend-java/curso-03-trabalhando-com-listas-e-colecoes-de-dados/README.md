# Curso 03 — Java: trabalhando com listas e coleções de dados

Este diretório reúne os materiais de estudo, exercícios e projetos do **terceiro curso** da carreira de Desenvolvedor Back-end Java na Alura.

O foco deste curso é aprender a trabalhar com listas e coleções de dados em Java, evoluindo o projeto ScreenMatch e praticando com desafios.

---

## Estrutura do curso

- `modulo-01-colecao-de-filmes/`
  Introdução ao ArrayList:
    - representação de listas de objetos com `ArrayList`;
    - manipulação com `add()`, `size()` e `get()`;
    - sobrescrita do método `toString()`.

- `modulo-02-construindo-objetos/`
  Construtores em Java:
    - passagem de parâmetros ao instanciar objetos;
    - construtor padrão (default);
    - declaração de construtores;
    - chamada do construtor da classe mãe com `super`.

- `modulo-03-lista-de-objetos-distintos/`
  Listas com tipos diferentes:
    - percorrer `ArrayList` com loop foreach;
    - casting de objetos entre tipos compatíveis;
    - verificação de tipo com `instanceof`.

- `modulo-04-ordenando-dados/`
  Ordenação de listas:
    - `Collections.sort` com interface `Comparable`;
    - implementação do método `compareTo`;
    - ordenação flexível com `Comparator`;
    - interface `List` e implementações `ArrayList` e `LinkedList`;
    - polimorfismo com `List`.

- `modulo-05-consolidando-conhecimentos/`
  Desafio final que aplica todos os conceitos do curso:
    - aplicação de cartão de crédito no console;
    - registro e validação de compras;
    - lista de compras ordenada por valor;
    - encapsulamento e organização por classes.

- `projeto-screenmatch/`
  Projeto ScreenMatch evoluído ao longo dos módulos:
    - `toString()` em `Filme` e `ArrayList` de títulos;
    - construtores em `Titulo`, `Filme` e `Serie`;
    - foreach com `instanceof` para cast em `Filme`;
    - ordenação com `Comparable` e `Comparator`.

- `docs/`
  Documentos relacionados ao curso (certificado de conclusão).

---

## Como executar os exercícios

1. Abra o projeto na sua IDE de preferência (IntelliJ IDEA, por exemplo).
2. Navegue até o módulo desejado dentro de `src/`.
3. Localize a classe que contém o método `main`.
4. Execute a aplicação pelo botão de **Run** da IDE.
