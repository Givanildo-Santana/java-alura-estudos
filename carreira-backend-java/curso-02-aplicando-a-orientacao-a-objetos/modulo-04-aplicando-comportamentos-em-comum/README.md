# Módulo 04 — Aplicando comportamentos em comum

Este módulo reúne os desafios práticos de interfaces do curso **"Java: aplicando a Orientação a Objetos"** da Alura.

---

## O que foi aprendido

- Criação e uso de interfaces com `interface` e `implements`;
- Diferença entre herança e interface;
- Polimorfismo com interfaces;
- Contrato entre classes sem herança direta;
- Aplicação de interfaces em contextos variados.

---

## Estrutura do módulo

Os desafios estão separados por pacote:

| Pacote | Interface | Classes | Descrição |
|---|---|---|---|
| `ex01_conversorMoeda` | `ConversaoFinanceira` | `ConversorMoeda` | Conversão de dólar para real |
| `ex02_CalculadoraSalaRetangular` | `CalculoGeometrico` | `CalculadoraSalaRetangular` | Área e perímetro de sala retangular |
| `ex03_TabuadaMultiplicacao` | `Tabuada` | `TabuadaMultiplicacao` | Exibe a tabuada de um número |
| `ex04_ConversorTemperatura` | `ConversorTemperatura` | `ConversorTemperaturaPadrao` | Conversão entre Celsius e Fahrenheit |
| `ex05_CalculadorTaxa` | `Calculavel` | `Livro`, `ProdutoFisico` | Preço final com desconto ou taxa |
| `ex06_CalculadorDesconto` | `Vendavel` | `Produto`, `Servico` | Preço total com desconto por quantidade |

---

## Como executar

1. Abra o projeto na sua IDE (IntelliJ IDEA, por exemplo).
2. Navegue até:

   `curso-02-aplicando-a-orientacao-a-objetos/modulo-04-aplicando-comportamentos-em-comum/src/`

3. Localize a classe `principal.br.com.givanildo.modulo02.desafios.Principal` dentro do pacote do exercício desejado.
4. Execute a aplicação (botão de **Run**).
