# Módulo 03 — Lista de objetos distintos

Este módulo apresenta como percorrer listas com objetos de tipos diferentes, utilizando casting e instanceof.

---

## O que foi aprendido

- Percorrer um `ArrayList` com loop foreach;
- Realizar casting de objetos, convertendo para outro tipo compatível;
- Verificar o tipo real de um objeto com `instanceof` antes de realizar o cast.

---

## Estrutura do módulo

- `projeto-screenmatch/` — `ArrayList<Titulo>` com `Filme` e `Serie`, foreach com `instanceof` para cast em `Filme`
- `desafios/` — exercícios práticos com ArrayList, herança, instanceof e interfaces

---

## Desafios

| Exercício | Classes | Descrição |
|---|---|---|
| `ex02_arraylist` | `Principal` | ArrayList de Strings percorrido com foreach |
| `ex03_animal_usando_heranca` | `Animal`, `Cachorro`, `Principal` | Upcasting de Cachorro para Animal |
| `ex04_usando_instanceof` | `Principal` | Downcasting de Animal para Cachorro com instanceof |
| `ex05_produtos_usando_arraylist` | `Produto`, `Principal` | Lista de produtos com cálculo de preço médio |
| `ex06_formas_geometricas_com_interface` | `Forma`, `Circulo`, `Quadrado`, `Principal` | Interface com calcularArea() e lista de formas |
| `ex07_conta_bancaria_com_arraylist` | `ContaBancaria`, `Principal` | Lista de contas bancárias encontrando a de maior saldo |

---

## Como executar

1. Abra o projeto na sua IDE (IntelliJ IDEA, por exemplo).
2. Navegue até:

   `curso-03-trabalhando-com-listas-e-colecoes-de-dados/src/`

3. Localize a classe `Principal` do pacote desejado.
4. Execute a aplicação (botão de **Run**).
