# Módulo 01 — Primeiro programa em Java

Este módulo é introdutório e não contém exercícios práticos em Java. O foco foi a configuração do ambiente e os conceitos fundamentais da plataforma Java.

---

## O que foi aprendido

- Configuração do ambiente de desenvolvimento com **JDK 17** e **IntelliJ IDEA**;
- Diferenças entre **JVM**, **JRE** e **JDK**:
    - **JVM** (Java Virtual Machine) — responsável por executar o bytecode;
    - **JRE** (Java Runtime Environment) — fornece as bibliotecas padrão para execução;
    - **JDK** (Java Development Kit) — compila o código-fonte em bytecode;
- Criação e execução do primeiro programa **"Alô Mundo!"** no IntelliJ;
- Entendimento do arquivo `.class` e o papel do bytecode na JVM.
