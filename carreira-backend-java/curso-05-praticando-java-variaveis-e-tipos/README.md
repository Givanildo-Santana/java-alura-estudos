# Curso 05 — Praticando Java: Variáveis e Tipos

> ⚡ **Curso prático** — Este não é um curso de conteúdo novo. É um complemento do
> Curso 01 (Java: criando a sua primeira aplicação), focado exclusivamente em
> exercícios e desafios para fixar variáveis, tipos primitivos, casting e condicionais.

---

## 🗂️ Estrutura do curso

- `modulo-01-variaveis-e-tipos/`
  10 desafios práticos cobrindo:
    - declaração e uso de variáveis (`String`, `int`, `double`, `boolean`, `char`);
    - conversão e casting entre tipos;
    - operações matemáticas e cálculo de médias;
    - estruturas condicionais (`if/else`);
    - resolução de problemas do cotidiano com lógica de programação.

- `docs/`
  Certificado de conclusão.

---

## ✏️ Desafios

| Exercício | Descrição |

| ex01 — Declarando valores a variáveis | Declarar e exibir variáveis dos tipos principais |
| ex02 — Conversão de tipos | Casting de `double` para `int` |
| ex03 — Primitivos em ação | Cálculo de média entre três notas |
| ex04 — Conversão de temperatura | Converter Celsius para Fahrenheit |
| ex05 — Cadastro de livros | Variáveis múltiplas e `char` com `if/else` |
| ex06 — Classificação por categoria | Categorizar produto por faixa de preço |
| ex07 — Par ou ímpar | Verificar paridade com operador `%` |
| ex08 — Conversor de moedas | Converter reais para dólares |
| ex09 — Verificação de idade | Escopo de variáveis e condicionais |
| ex10 — Planejamento de viagem | Autonomia de veículo com `printf` |
