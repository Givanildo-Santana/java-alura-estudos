# Curso 01 — criando a sua primeira aplicação

Este diretório reúne os materiais de estudo, exercícios e desafios do **primeiro curso** da carreira de Desenvolvedor Back-end Java na Alura.

O foco deste curso é dar os primeiros passos na linguagem, escrevendo programas simples de console, trabalhando com variáveis, entrada de dados e estruturas de controle.

## Estrutura do curso

Dentro de `curso-01-criando-sua-primeira-aplicacao/` você encontrará:

- `modulo-01-fundamentos/`  
  Conceitos básicos da linguagem:
    - tipos primitivos e variáveis;
    - operações aritméticas;
    - conversão de tipos (casting);
    - uso básico de `String`;
    - primeiras impressões no console com `System.out.println` / `System.out.printf`.

- `modulo-02-condicionais-e-loops/`  
  Controle de fluxo e interação com o usuário:
    - leitura de dados com `Scanner`;
    - estruturas condicionais `if`, `if/else`;
    - laços de repetição `for` e `while`;
    - pequenos programas com menus no console, verificações (par/ímpar, maior/menor, etc.) e repetições.

- `desafio-conta-bancaria/`  
  Desafio prático que consolida os conteúdos do curso:
    - simulador de conta bancária no console;
    - consulta de saldo, depósitos e transferências;
    - validação de valores e atualização de saldo;
    - menu interativo em loop.
  > Detalhes do desafio estão no `README.md` dentro da pasta `desafio-conta-bancaria/`.

- `docs/`  
  Documentos relacionados ao curso (por exemplo, certificado de conclusão).

## Como executar os exercícios

1. Abra o projeto na sua IDE de preferência  
   (IntelliJ IDEA, Eclipse, VS Code com extensão de Java, etc.).

2. Navegue até o módulo desejado, por exemplo:

    - `modulo-01-fundamentos/src/`
    - `modulo-02-condicionais-e-loops/src/`
    - `desafio-conta-bancaria/src/`

3. Localize a classe que contém o método `main`.

4. Execute a aplicação pela IDE (botão de “Run”)
