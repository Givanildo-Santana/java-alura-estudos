# Curso 02 — Pensamento Computacional: fundamentos da computação e lógica de programação

Curso teórico de fundamentos da computação e lógica de programação. Sem código Java — focado em algoritmos, pseudocódigo e pensamento computacional.

## Conteúdo

- **Mundo da Computação** — fundamentos, hardware, software e economia criativa
- **Explorando o pensamento computacional** — decomposição, padrões, abstração e algoritmos
- **Resolvendo problemas** — algoritmos, modularidade e decisão
- **Lógica de Programação** — variáveis, operadores, funções e estruturas
- **Mundo da programação** — linguagens, compiladores, terminal e IA

## Conceitos praticados

- Pensamento computacional
- Decomposição de problemas
- Reconhecimento de padrões
- Abstração
- Algoritmos e fluxogramas
- Lógica de programação (pseudocódigo)
