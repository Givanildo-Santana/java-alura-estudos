# Módulo 01 — Coleção de filmes

Este módulo apresenta o uso de listas em Java com a classe `ArrayList`, aplicado ao projeto ScreenMatch e em exercícios práticos.

---

## O que foi aprendido

- Representação de listas de objetos com `ArrayList`;
- Manipulação com os métodos `add()`, `size()` e `get()`;
- Diferentes tipos de classes do Java para representar listas;
- Sobrescrita do método `toString()` para representar objetos em texto.

---

## Estrutura do módulo

- `projeto-screenmatch/` — projeto ScreenMatch evoluído com `ArrayList<Filme>` e `toString()` na classe `Filme`
- `desafios/` — exercício prático com `Pessoa`, `ArrayList` e `toString()`

---

## Desafio

| Pacote | Classes | Descrição |
|---|---|---|
| `modulo01.desafios` | `Pessoa`, `principal.br.com.givanildo.modulo02.desafios.Principal` | Cria lista de pessoas, exibe tamanho, primeiro elemento e lista completa |

---

## Como executar

1. Abra o projeto na sua IDE (IntelliJ IDEA, por exemplo).
2. Navegue até:

   `curso-03-trabalhando-com-listas-e-colecoes-de-dados/src/`

3. Localize a classe `principal.br.com.givanildo.modulo02.desafios.Principal` do pacote desejado.
4. Execute a aplicação (botão de **Run**).
