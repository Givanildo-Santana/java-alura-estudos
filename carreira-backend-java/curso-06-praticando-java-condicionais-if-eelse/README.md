# Curso 06 — Praticando Java: Condicionais if e else

> ⚡ **Curso prático** — Este não é um curso de conteúdo novo. É um complemento do
> Curso 01 (Java: criando a sua primeira aplicação), focado exclusivamente em
> exercícios e desafios para fixar condicionais `if`, `else if`, `else` e operadores
> lógicos e relacionais.

---

## 🗂️ Estrutura do curso

- `modulo-01-if-else/`
  10 desafios práticos cobrindo:
    - condicionais `if`, `else if` e `else`;
    - operadores relacionais (`>`, `<`, `==`, `>=`, `<=`);
    - operadores lógicos (`&&`, `||`, `!`);
    - leitura de dados com `Scanner`;
    - resolução de problemas do cotidiano com lógica de programação.

- `docs/`
  Certificado de conclusão.

---

## ✏️ Desafios

| Exercício | Descrição |
|---|---|
| ex01 — Verificando se um número é par ou ímpar | Operador `%` com condicional |
| ex02 — Verificando aprovação em uma disciplina | Três faixas de resultado com `else if` |
| ex03 — Validação de senha | Comparação de Strings com `equals()` e `Scanner` |
| ex04 — Comparando dois números | Leitura de dois valores e comparação |
| ex05 — Verificação de desconto em compras | Desconto condicional com `printf` |
| ex06 — Verificação de dia útil | Operador `\|\|` para múltiplas condições |
| ex07 — Verificando se um número está em um intervalo | Operador `&&` para intervalo |
| ex08 — Verificação de triângulo | Condição com três lados simultâneos |
| ex09 — Verificação de compatibilidade de doação de sangue | Variáveis booleanas e motivos detalhados |
| ex10 — Verificação de código de acesso e nível de permissão | Combinação de código e nível com `&&` |
