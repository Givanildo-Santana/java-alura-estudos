# Módulo 03 — Reaproveitando características e comportamentos

Este módulo reúne os desafios práticos de herança e polimorfismo do curso **"Java: aplicando a Orientação a Objetos"** da Alura.

---

## O que foi aprendido

- Herança entre classes com `extends`;
- Reaproveitamento de comportamentos;
- Polimorfismo com `@Override`;
- Modificador `protected`;
- Uso de `super`.

---

## Estrutura do módulo

Os desafios estão separados por pacote:

| Pacote | Classes | Descrição |
|---|---|---|
| `ex01_carro` | `Carro`, `CarroAvaliado` | Herança com histórico de preços e cálculo de menor/maior preço |
| `ex02_animal` | `Animal`, `Cachorro`, `Gato` | Herança com `emitirSom()`, `locomover()` e comportamentos específicos |
| `ex03_contaBancaria` | `ContaBancaria`, `ContaCorrente` | Herança com `cobrarTarifaMensal()` reutilizando `sacar()` |
| `ex04_numerosPrimos` | `NumerosPrimos`, `VerificadorPrimo`, `GeradorPrimo` | Herança com verificação e geração de números primos |

---

## Como executar

1. Abra o projeto na sua IDE (IntelliJ IDEA, por exemplo).
2. Navegue até:

   `curso-02-aplicando-a-orientacao-a-objetos/modulo-03-reaproveitando-caracteristicas-e-comportamentos/src/`

3. Localize a classe `principal.br.com.givanildo.modulo02.desafios.Principal` dentro do pacote do exercício desejado.
4. Execute a aplicação (botão de **Run**).
